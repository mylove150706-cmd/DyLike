# DyLike 用户指南

`DyLike` 是一个以 `dy-player` 为主入口的 Android 视频播放器。它支持本地播放、网络资源浏览、WebDav、串流链接、字幕、弹幕、媒体库和播放历史等能力。  

项目主页及下载链接：https://zhaohuaxs.github.io/dy-like.html  

## 你可以用它做什么

- 播放本地视频文件  
- 通过文件管理器或其他应用直接打开视频  
- 通过串流链接直接播放在线视频  
- 浏览 WebDav 资源并直接进入播放  
- 使用长视频模式观看剧集、电影、番剧等内容  
- 使用短视频模式连续滑动播放内容  
- 使用字幕功能和部分字幕轨道能力  
- 在长视频场景中加载和显示弹幕  
- 管理历史记录、收藏和媒体库  
- 备份和恢复应用数据  

## 核心特性

- 支持本地媒体库与在线媒体库  
- 支持本地存储、外部存储、WebDav、串流链接等资源入口  
- `dy-player` 支持 `Exo` / `MPV` 播放内核切换  
- 支持字幕加载  
- 支持弹幕显示、样式调整、字体切换和合并显示  
- 支持自动播放下一集  
- 支持片头 / 片尾偏移配置  
- 支持桌面图标切换  
- 支持应用数据备份与恢复  

## 快速开始

1. 安装并启动 `dy-player`  
2. 首次进入时授予存储和网络相关权限  
3. 在 `资源库` 中添加或打开你的资源来源  
4. 在 `媒体库` 中创建媒体库，或直接扫描本地视频  
5. 根据内容场景选择长视频模式或短视频模式  
6. 在 `我的` 页面中调整播放器、弹幕和备份等设置  

## 支持的内容来源

- 本地视频文件  
- HTTP / HTTPS 视频链接  
- WebDav 远程资源  

## 页面说明

### 媒体库

适合管理“已经整理好的内容集合”，例如番剧、剧集、电影片库、收藏和播放历史。  
这里可以添加本地媒体库、在线媒体库，也可以进入媒体详情页统一管理剧集与播放进度。  

### 资源库

适合浏览“原始资源来源”，例如手机本地目录、外部存储、WebDav 或一条可直接播放的串流链接。  
如果你更习惯“先找文件，再播放”，这个页面会更顺手。  

### 我的

适合管理播放器行为和个性化设置，包括：  

- 播放器内核与渲染方式  
- 弹幕显示和样式  
- 数据备份与恢复  
- 图标切换  
- 关于信息
